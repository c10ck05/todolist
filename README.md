# 스터디 플래너 (투두리스트)

내가 할 일을 적고 관리할 수 있는 간단한 투두리스트 웹사이트 프로젝트입니다. 로그인과 회원가입 기능이 있고, 보안이랑 이메일 인증 기능도 넣었습니다.

---

## 1. 주요 기능

* **회원가입이랑 로그인**
  * 회원가입을 하거나 비밀번호를 바꿀 때 진짜 내 이메일로 6자리 인증번호가 날아옵니다. (3분 제한)
  * 비밀번호는 해킹당하지 않게 암호화되어서 안전하게 저장됩니다.
  * 로그인하면 나만 글을 쓰고 지울 수 있게 토큰(JWT)이라는 인증 열쇠를 받습니다.

* **할 일 관리하기 (투두리스트)**
  * 로그인한 사람마다 자기만의 투두리스트를 따로 가집니다.
  * 새로운 할 일을 등록하고, 필요 없어진 일은 X 버튼을 눌러서 지울 수 있습니다.

---

## 2. 테이블 구성 (데이터베이스 스키마)

### users (사용자 테이블)
* `id` (Integer, PK): 고유 식별 번호 (자동 증가)
* `user_id` (String, Unique): 사용자 로그인 아이디
* `password` (String): 암호화된 비밀번호 해시 값
* `email` (String, Unique): 인증 및 비번 재설정용 이메일 주소

### todolist (할 일 테이블)
* `id` (Integer, PK): 투두 고유 번호 (자동 증가)
* `todo` (Text): 할 일 내용 기입란
* `owner_id` (String): 작성자의 user_id (사용자 식별용)
* `completed` (Boolean): 완료 여부 상태값 (Default: False)

### email_verifications (이메일 인증 테이블)
* `email` (String, PK): 인증을 요청한 이메일 주소
* `code` (String): 발송된 6자리 인증 번호
* `expires_at` (DateTime): 인증 번호 만료 시간 (생성 후 3분)

---

## 3. 화면 구성

컴퓨터 화면 하나(`index.html`) 안에서 버튼을 누를 때마다 아래 4개 화면이 번갈아가면서 나타납니다.

* **로그인 화면**
  * 아이디랑 비밀번호를 입력하고 로그인하는 화면입니다. 회원가입이나 비밀번호 찾기로 넘어가는 링크가 있습니다.

* **회원가입 화면**
  * 사용할 아이디, 비밀번호를 적고 이메일 인증을 받아서 가입을 완료하는 화면입니다.

* **비밀번호 재설정 화면**
  * 비밀번호를 까먹었을 때 이메일 인증을 다시 받아서 새 비밀번호로 바꾸는 화면입니다.

* **투두리스트 메인 화면**
  * 로그인을 성공하면 나오는 화면입니다. 할 일을 입력해 등록할 수 있습니다.
