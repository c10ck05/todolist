# Study Planner (Todo List Web Service)

FastAPI와 가벼운 정적 HTML 구조를 기반으로 작동하는 사용자 맞춤형 투두리스트 서비스입니다. JWT를 활용한 보안 인증과 이메일을 통한 검증 시스템을 포함하고 있습니다.

---

## 1. 핵심 기능

* **사용자 인증 및 보안**
  * 회원가입 및 비밀번호 재설정 시 Gmail SMTP 기반 6자리 이메일 인증 (3분 제한)
  * Bcrypt 라이브러리를 활용한 비밀번호 단방향 암호화 저장
  * 로그인 성공 시 유저 식별용 JWT(JSON Web Token) 발급 및 로컬 스토리지 저장 관리

* **투두리스트 관리 (CRUD)**
  * JWT 토큰 검증을 통한 사용자별 독립적인 투두 데이터 격리 및 관리
  * 할 일 등록, 조회, 삭제 기능
  * 할 일 완료 상태 토글(수정) 기능 및 완료 항목 별도 필터링 분리

---

## 2. 테이블 구성 (데이터베이스 스키마)

### users (사용자 테이블)
* `id` (Integer, PK): 고유 식별 번호 (자동 증가)
* `user_id` (String, Unique): 사용자 로그인 아이디
* `password` (String): 암호화된 비밀번호 해시 값
* `email` (String, Unique): 인증 및 비번 재설정용 이메일 주소

### todolist (할 일 테이블)
* `id` (Integer, PK): 투두 고유 번호 (자동 증가)
* `todo` (Text): 할 일 내용 기입란
* `owner_id` (String): 작성자의 user_id (사용자 식별용)
* `completed` (Boolean): 완료 여부 상태값 (Default: False)

### email_verifications (이메일 인증 테이블)
* `email` (String, PK): 인증을 요청한 이메일 주소
* `code` (String): 발송된 6자리 인증 번호
* `expires_at` (DateTime): 인증 번호 만료 시간 (생성 후 3분)

---

## 3. 화면 구성 (구획 인터페이스)

단일 정적 페이지(`index.html`) 내에서 조건부 상태 관리를 통해 총 4개의 화면 구획이 유기적으로 전환됩니다.

* **로그인 화면 (`loginSection`)**
  * 아이디 및 비밀번호 입력 폼
  * 회원가입 화면 및 비밀번호 재설정 화면 이동 링크

* **회원가입 화면 (`registerSection`)**
  * 아이디, 비밀번호, 비밀번호 확인 입력 폼
  * 이메일 입력 및 인증번호 요청 버튼, 인증번호 입력 폼
  * 최종 회원가입 완료 버튼

* **비밀번호 재설정 화면 (`resetSection`)**
  * 가입된 이메일 입력 및 인증코드 요청 버튼
  * 인증번호 입력 및 변경할 새 비밀번호 입력 폼
  * 비밀번호 최종 변경 버튼

* **나의 투두리스트 메인 화면 (`todoSection`)**
  * 로그아웃 버튼 및 상단 할 일 신규 등록 폼
  * **필터 탭**: [전체] / [할 일] / [완료] 상태별 리스트 필터링 토글 버튼
  * **투두 아이템 리스트**: 체크박스(완료 토글)와 텍스트, 삭제(X) 버튼으로 구성
